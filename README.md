# GoalAssistant(가제)

## 소개
> 공부를 하고싶다고요? 도서관에서 앱을 차단해 보세요.

> 장소를 지정하고, 해당 장소에서 차단하려는 앱을 선택할 수 있습니다.

> 자신의 위치가 지정한 장소 근처일 경우, 앱 실행을 차단합니다.

### 순한맛

> 지정 장소와 내 위치가 가까워도 GoalAssistant 앱에 접근이 가능합니다

> 앱 차단 해제가 가능합니다
